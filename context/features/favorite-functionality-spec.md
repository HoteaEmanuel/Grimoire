# Favorite Functionality

## Overview

Add the favoriting functionality to items and collections, faovriting should pe on the drawer,collection page, cards and anywhere else the item is displayed. The user should be able to favorite and unfavorite items and collections, and the favorites page should display all favorited items and collections.
