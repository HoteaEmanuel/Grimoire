# Add pro sidebar badge to sidebar


# Overview
Add a pro badge to the files and the images type in the sidebar

# Requirements
- Add a pro badge to the sidebar item types "Files" and "Images" using the shadcn badge component
- The badge should be a small circle with "PRO" text inside, positioned to the right of the item type name
- Use a distinct color for the badge (e.g. red or blue) to make it stand out
- Ensure the badge is responsive and looks good on different screen sizes
- Make Pro uppercase and use a bold font weight for the text inside the badge
- Make sure the badge is clean and subtle, not overpowering the item type name but still noticeable
- The badge should be added only to the "Files" and "Images" item types in the sidebar, not to the other item types
